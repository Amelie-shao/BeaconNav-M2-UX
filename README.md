# Overview
BeaconNav is a user-centred indoor navigation prototype designed for AUT’s WG building.  
It helps students locate classrooms quickly by integrating timetables, accessibility settings, and offline maps.  
This repository contains only UI/UX design artefacts and evidence — not source code.

# Evaluation Summary
Usability tested with 5 AUT students using the PACMAD model (Harrison et al., 2013).  
Results showed Excellent usability (SUS = 82.4) and high efficiency (average task time = 92s).  
Full evaluation details are included in the Milestone 2 report submitted on Canvas.
